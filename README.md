[TensorFlow Datasets](https://www.tensorflow.org/datasets) for the iCyt platform.

## Installation

1. Download the `.tar.gz` dataset and copy it to `~/tensorflow_datasets/downloads/manual/`.
2. Clone this repository and execute `tfds build` from the `poldiv` directory. The installation can take several hours.

## Usage

```python
import tensorflow as tf
import tensorflow_datasets as tfds

(ds_train, ds_test), ds_info = tfds.load('poldiv/all:1.0.0', split=['train[20%:]','train[:20%]'], shuffle_files=True, with_info=True)
assert isinstance(ds_train, tf.data.Dataset)
assert isinstance(ds_test, tf.data.Dataset)
print(ds_info)
```
